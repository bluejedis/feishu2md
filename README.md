本仓库对应windows amd64，直接下载PATH文件夹，并在高级电脑设置中 添加此路径为新的环境变量

- 【自己遇到的问题 是
  - 测试端，一定要写版本发布
  - 管理员，要审批权限
  - id和密码的输入，还是要用此文档中的。而不是用vscode打开修改，并没有修改成功
    - `feishu2md config --appId <your_id> --appSecret <your_secret>` 
      - 其中<your_id>的"<>"只是占位符，输入时，只需要输入自己的id和密码即可】
